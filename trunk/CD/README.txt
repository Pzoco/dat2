To see the source-code via Visual Studio:

1. Install the XNA framework:  
( http://www.microsoft.com/downloads/en/details.aspx?familyid=a88c6dec-aeae-42cd-a108-d35c013c3b97 )
2. Open up the "Source/WarSimulator.sln" file


View the report as a PDF file:

1. Get a PDF Reader
( www.adobe.com )
2. Open the Report/report.pdf


Run the binaries:

1. Open "Binaries/WarSimulator.exe"

The simulation will then run with our TestData.



To create your own script, run the simulator via CommandPrompt.

# WarSimulator.exe <configfile> <teamfile 1> <teamfile 2> ... <teamfile n>
